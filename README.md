# Machine-Learning

Repositori ini berisi pipeline lengkap untuk proyek Machine Learning yang mencakup:

## 1. Fitur Utama
- Serialisasi data menggunakan LLM
- Anonimisasi teks (Hybrid Anonymization)
- Preprocessing NLP
- Weak Labelling QI
- Ekstraksi fitur (BoW, TF-IDF, Bigram)
- Model Machine Learning klasik
- Model Deep Learning (RNN, LSTM, FT-LSTM, LoRA-BERT)
- Analisis eror model
